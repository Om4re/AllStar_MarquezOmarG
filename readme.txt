#Como se puede ver en el codigo se intento hacer de la manera mas sencilla posible siguiendo como ejemplo todas las clases que llevamos hasta ahora.
En la pantalla de views se almacenamos todas las funciones
En la pagina path la usamos para definir todas las rutas que vinculamos con las funciones
En la pagina models la utilizamos para definir y gestionar nuestras clases 
La pagina forms se uso para la creacion de nuevas clases
La pagina admin se uso para registrar los modelos que usamos para la creacion de la pagina web
